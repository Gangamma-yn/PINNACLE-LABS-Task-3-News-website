# 📰 Global News Hub

A modern and responsive News Aggregator Website that brings together the latest headlines from around the world. The platform provides users with quick access to news across multiple categories including Technology, Business, Sports, Health, Entertainment, Science, and World News.

## 🚀 Project Overview

Global News Hub is designed to serve as a one-stop destination for reading current news from various global sources. The website fetches real-time news using a News API and displays articles in an organized, user-friendly interface.

## ⚙️ Installation and Setup

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/global-news-hub.git
```

### 2. Open the Project Folder

```bash
cd global-news-hub
```

### 3. Get a News API Key

Create a free account at:

https://newsapi.org

Copy your API key.

### 4. Update API Key

Open `script.js` and replace:

```javascript
const API_KEY = "YOUR_API_KEY";
```

with:

```javascript
const API_KEY = "YOUR_ACTUAL_API_KEY";
```

### 5. Run the Project

Open `index.html` using Live Server in Visual Studio Code.

Or simply open the file in your browser.
